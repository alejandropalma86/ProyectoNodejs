Bienvenidos

En este proyecto se utiliza NodeJS y MongoDB para la creación de un CRUD que es el acrónimo de "Crear, Leer, Actualizar y Borrar" (del original en inglés: Create, Read, Update and Delete), que se usa para referirse a las funciones básicas en bases de datos o la capa de persistencia en un software.
